# Pharmacy Chain Data Simulation — SQL / Python / Tableau

End-to-end data project simulating a 1,000-store pharmacy retail chain over
20 years of operation. Covers the full pipeline: relational schema design
(22 tables — dimensions, transaction facts, snapshot facts, and aggregate
facts), SQL Server implementation with views/stored procedures/indexing
strategy (including columnstore), 200+ solved analytical SQL questions
(from basic joins to RFM segmentation and cohort analysis), Python analysis
scripts, and a Tableau dashboard plan with a working HTML prototype.

**Stack:** SQL Server (T-SQL) · Python (pandas, SQLAlchemy) · Tableau
